int TOOL::determineApp ( Netflow *  n,

)

Identifies application, which has opened socket which belongs to some IP, proto and port.

In case it is called with Netflow which is already in cache, but application has changed, old record is copied into g_finalResults vector. Update mode means that instead of moving 'n' into 'e', we just update times in 'e'

Parameters
[in]nNetflow information
[out]eSet application and socket inode number with netflow structure
[in]modeUpdate of expired record or inserting new record
Returns
Value bigger than zero if there wasn't any error. -1 is returned if application or inode wasn't found - in this case TOOL::TEntry::appName is set to empty string. If there were any Input/Output error, -2 is returned.
